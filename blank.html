A Rundown of Skills I've Acquired on My Journey

I get asked a lot about what it is exactly that I do, since software engineering is rather broad. I'm certainly not a jack of all trades, but here’s a list of all the skills I've acquired through the years.

Android

Kotlin

I've been creating Android applications with Kotlin for 3+ years. My published Android application Ramanie has been built entirely using Kotlin.

XML

XML has been at the core of my Android projects; I've built complex user interfaces with it for more than half a decade.

Jetpack

This new suite of libraries from Google has improved my workflows.

Room

An Android library that helps me manage the local database.

Architecture

I work with architecture patterns such as MVVM and MVI.

Web

Typescript

I prefer creating with TypeScript, which is a superset of JavaScript.

Angular

I started building web applications with Angular ~3 years ago.

React JS

I've gained experience in building robust front-end web apps and sites with React.

Tailwind CSS

I use Tailwind in most of my web projects to create stunning user interfaces.

Design

Wireframing

I use wireframes to visualize project objectives and user journeys.

Figma

Figma is a great tool for creating high-fidelity wireframes.

Languages

  • English: Native speaker with C2 proficiency.
  • Español: B1 intermediate speaker.
  • Français: A1 level of proficiency.
  • Swahili: B1 level proficiency.
